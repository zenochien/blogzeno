Learning Keywords

October, 11110 One-minute read

The C# language reserves 77 identifiers, you learn keywords remember!!

Examples of keywords are class, namespace, and using. You’ll learn the meaning of most of the C# keywords as you proceed through this book. The following is the list of keywords:

C# also uses the following identifiers. These identifiers are not reserved by C#, which means that you can use these names as identifiers for your own methods, variables, and classes, but you should avoid doing so if at all possible.

Working with primitive data types

The numeric types of C#, together with other common types for holding strings, characters, and Boolean values, are collectively known as the primitive data types.
